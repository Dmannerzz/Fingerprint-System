# Fingerprint Recognition System

This project was designed and implemented as part of the **CPE309 course**.  
It demonstrates the use of Arduino hardware and fingerprint sensor modules to build a biometric authentication system.

## Project Overview
- **Course**: CPE309  
- **Objective**: To design a secure fingerprint-based access system using Arduino.  
- **Key Features**:
  - Fingerprint enrollment and verification
  - Secure access control logic
  - Modular design for easy integration with other systems

## File Information
- `Fingerprint_System_2.1.ino` → Main Arduino sketch containing the fingerprint recognition logic.

## How to Use
1. Upload the `.ino` file to your Arduino board using the Arduino IDE.
2. Connect the fingerprint sensor module as specified in the code.
3. Enroll fingerprints and test the authentication process.
